JavaScript数据结构——集合

2016.05.13 Justin Wong

集合是一个数学概念，是由0~N个无序且唯一的项组成的数据结构。可以简单地把集合看做成一个既没有重复元素也没有顺序概念的数组。

集合的实现

对于集合的实现，可以使用数组或对象保存集合中的元素，在下面的实现中采用对象表示集合。

对于集合以及集合操作需要实现以下方法：

  • add(value)：向集合中添加一个新元素；
  • delete(value)：从集合中移除该元素；
  • has(value)：如果元素在集合中则返回true，否则返回false；
  • clear()：移除集合中的所有元素；
  • size()：返回集合中的元素数量；
  • values()：返回整个集合的值（数组形式）；
  • union(otherSet)：求并集，返回运算结果；
  • intersection(otherSet)：求交集，返回运算结果；
  • difference(otherSet)：求差集，返回运算结果；
  • subset(otherSet)：判断是否是otherSet的子集。

ES6中的Set类

ES6提供了一个原生的Set类来表示集合这个数据结构。使用ES6的Set需要特别注意浏览器兼容性问题。

构造函数 new Set([iterable])：传入一个可迭代对象（如Array、String等），其中的所有元素将被加入到Set中。注意Set中的元素不能重复。

Set中元素的值是唯一的，但判断值是否相等的算法和严格相等不完全一致，主要区别在于NaN !== NaN的结果为true，但在Set中NaN被视为相同的值。

所有Set实例都继承自Set.prototype，Set.prototype的方法有：

  • Set.prototype.add(value)：向Set中添加一个元素并返回Set对象；
  • Set.prototype.delete(value)：移除Set中该元素并返回结果；
  • Set.prototype.has(value)：返回一个布尔值，表示该元素是否存在于Set中；
  • Set.prototype.clear()：移除Set中的所有元素；
  • Set.prototype.keys()：返回一个键名的迭代器；
  • Set.prototype.values()：返回一个键值的迭代器；
  • Set.prototype.entries()：返回一个键值对的迭代器；
  • Set.prototype.forEach(callbackFn[, thisArg])：使用回调函数按插入顺序遍历Set中每个元素。
