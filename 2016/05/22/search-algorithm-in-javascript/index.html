JavaScript算法——搜索

2016.05.22 Justin Wong

搜索算法有很多种，在这里只关注两种最基本的搜索算法：顺序搜索和二分搜索。

顺序搜索

顺序搜索/线性搜索，是一种最基本、很低效的搜索算法。它的原理是将每一个数据结构中的元素和我们要找的元素作比较。

从代码可以看出，顺序搜索的时间复杂度是O(n)。

二分搜索

二分搜索/折半搜索的效率比顺序搜索高。它的原理是每次折半，看要搜索的数和中间值的大小关系。但二分搜索要求数据结构是已排序的。

二分搜索的时间复杂度是（不考虑排序）：O(logn)。

除了顺序搜索和二分搜索外，之前的DFS和BFS也是搜索算法。
