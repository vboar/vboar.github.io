JavaScript数据结构——链表

2016.05.12 Justin Wong

和数组类似，链表是一个存储有序元素的集合。但链表和数组的区别在于：

  • 链表中的元素在内存中不是连续放置的；
  • 在链表中添加或移除元素不需要移动其他元素；
  • 不能直接访问链表中的某一元素，需要从链表头部开始迭代。

单向链表

需要一个Node类来表示链表的结点，链表中首先需要包含一个指向头结点的指针。
。
链表需要实现以下比较常用的方法：

  • insertFirst(element)：在链表头部添加一个新元素；
  • insertLast(element)：在链表尾部添加一个新元素；
  • insert(position, element)：在链表特定位置插入一个新元素并返回结果；
  • indexOf(element)：返回该元素在链表中的索引，若找不到该元素则返回-1；
  • removeHead()：移除链表头部元素并返回该元素；
  • removeTail()：移除链表尾部元素并返回该元素；
  • remove(element)：从链表中移除该元素并返回该元素；
  • removeAt(position)：移除链表中特定位置的元素并返回该元素；
  • isEmpty()：判断链表是否为空；
  • size()：返回链表包含的元素个数；
  • getHead()：返回链表头部节点；
  • getTail()：返回链表尾部节点；
  • displayFromHead()：从链表头部开始遍历

双向链表

双向链表和单向链表的区别在于：

  • 双向链表增加了一个指向尾部结点的指针；
  • 双向链表利用上述指针很容易实现从链表尾部开始向前遍历；
  • 双向链表中的每个结点增加了一个指向其前驱结点的指针，因此占用空间更大一些。

对于双向链表，需要额外多实现的方法有：

  • lastIndexOf(element)：从链表尾部开始查找，返回该元素在链表中的索引，若找不到该元素则返回-1；
  • displayFromTail()：从链表尾部开始遍历
