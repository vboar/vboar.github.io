JavaScript数据结构——栈

2016.05.02 Justin Wong

栈是一种后进先出（Last In Frist Out, LIFO）的有序集合。

用JavaScript实现栈

可以用数组来表示栈中元素。
需要为栈实现以下方法：

  • push(element)：添加一个新元素到栈顶
  • pop()：移除栈顶元素并返回该元素
  • peek()：返回栈顶元素（不修改栈）
  • isEmpty()：判断栈是否为空
  • clear()：移除栈中所有元素
  • size()：返回栈中元素个数
