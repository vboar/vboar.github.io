JavaScript数据结构——数组

2016.05.02 Justin Wong

数组（Array）是非常常见的数据结构，ECMAScript中的数组和其他语言不同的地方在于：

  • 数组中的每一项可以保存任何类型的数据
  • 数组的大小是可以动态调整的，即随着数据的的添加自动增长

检测数组

结合 JavaScript中的变量 中的类型检测来看。

检测数组一般有以下三种方法：

  • instanceof
  • Array.isArray()
  • Obejct.prototype.toString()

instanceof：

如果只有一个全局执行环境，使用instanceof可以得到正确的结果。但如果网页包含多个框架，即存在多个不同的全局环境，那么就会存在多个不同版本的Array构造函数。如果从一个框架向另一个框架传入数组，那么传入的数组和第二个框架中原生创建的数组具有不同的构造函数，无法判断是否Array类型。

Array.isArray()：

该方法可以解决上述instanceof存在的问题。一些老版本的浏览器尚未实现这个方法。

Obejct.prototype.toString()：

全浏览器通用，对于未实现Array.isArray方法的浏览器可以使用该方法来实现兼容。

数组的转换方法

toString方法：调用数组中每一项的toString方法，然后默认以逗号分隔开每一项，形成字符串并返回。
toLocaleString方法：调用数组中每一项的toLocaleString方法，然后默认以逗号分隔开每一项，形成字符串并返回。
valueOf方法：返回数组本身。
join方法：接收一个参数，即用作分隔符的字符串，然后返回包含所有数组项的字符串。

数组的重排序方法

数组中有两个重排序方法（改变原来的数组）：

  • reverse()：反转数组项的顺序并返回排序之后的数组
  • sort()：默认调用每个数组项的toString方法，然后比较字符串，按升序排列数组项并返回排序之后的数组

见上述例子，一般不会使用sort方法的不传参用法。
sort方法接收一个比较函数作为参数，以便指定哪个值位于哪个值的前面。
比较函数接收两个参数，如果第一个参数位于第二个之前则返回一个负数，相等放回0，之后返回一个正数。（升序）

数组的操作方法

数组的操作方法有以下三个：

  • concat()
  • slice()
  • splice()

concat()：
concat方法主要用于拼接数组，可以基于当前数组中的所有项创建一个新数组。
concat方法会先创建当前数组的一个副本，然后将接收到的参数添加到这个副本的末尾，最后返回新构建的数组。
concat方法的参数是可选的，参数可以是数组也可以是其他任意类型的值。
concat方法不会影响原始数组。

slice()：
slice方法主要用于数组截取，可以基于当前数组中的一个或多个项创建一个新数组。
slice方法接收一个或两个参数。
一个参数：作为返回项的起始位置，结束位置在数组末尾。
两个参数：作为返回项的起始和结束位置（返回数组不包括结束位置的项）。
slice方法的参数中，若结束位置小于起始位置，则返回空数组。
slice方法不会影响原始数组。

如上述例子，如果slice方法的参数是负数，则用负数加上数组长度来确定位置。

splice()：
splice方法可以在数组的任意位置插入、删除或替换数组项：

  • 删除：接收两个参数，第一个是要删除的第一项的位置，第二个是要删除的项数
  • 插入：接收三个或多个参数，第一个是起始位置，第二个是0（即要删除的项数），第三个和后面的参数都是要插入的项
  • 替换：接收三个或多个参数，第一个是起始位置，第二个是要删除的项数，第三个和后面的参数都是要插入的项

总的来说，splice方法接收的参数如下：

  • 起始位置
  • 要删除的项数
  • 0~N个参数：要插入的每一项

splice方法的返回值是一个数组，数组包含从原始数组中删除的项（没有删除项则返回空数组）。
splice方法会影响原始数组。

数组的位置方法

数组的位置方法有两个：

  • indexOf()：从数组开头开始向后查找
  • lastIndexOf()：从数组末尾开始向前查找

上述两个方法都接收两个参数，第一个是要查找的项，第二个是查找起点位置的索引（可选）。
返回值是要查找的项在数组中的位置，如没找到返回-1。
比较第一个参数与数组中每一项时，使用全等操作符。

数组的迭代方法

数组的迭代方法有五个，每个方法都接收两个参数，第一个是要在每一项上运行的函数，第二个是运行该函数的作用域对象（可选）。
传入这些方法中的函数接收三个参数：数组项的值，该项在数组中的位置，数组对象本身。

五个迭代方法如下：

  • every()：如果函数对每一项都返回true，则结果返回true
  • filter()：返回该函数会返回true的项组成的新数组
  • forEach()：每一项运行函数，没有返回值
  • map()：返回每次函数调用结果组成的新数组
  • some()：如果函数对任一项返回true，则结果返回true

数组的归并方法

数组的归并方法有两个：

  • reduce()：从数组的第一项开始，逐个遍历到最后
  • reduceRight()：从数组的最后一项开始，逐个遍历到第一项

这两个方法都会迭代数组的所有项，然后构建一个最终返回的值。
接收两个参数，第一个是在每一项上调用的函数，第二个是作为归并基础的初始值（可选）。
传入的函数接收四个参数：前一个值、当前值、项的索引、数组对象。函数的返回值会作为下一项的第一个参数。
第一次迭代发生在数组的第二项上，因此这次迭代的第一个参数是数组的第一项，第二个参数是数值的第二项。
