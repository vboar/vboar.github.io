JavaScript数据结构——字典

2016.05.15 Justin Wong

字典以键值对的形式来存储元素，其中键是唯一的，每个键对应一个唯一的值。字典可以称为映射。

字典的实现

对于字典需要实现以下方法：

  • set(key, value)：向字典中添加新元素；
  • get(key)：通过键名返回对应的值；
  • has(key)：如果键在字典中则返回true，否则返回false；
  • delete(key)：从字典中移除某个键；
  • clear()：移除字典中所有元素；
  • size()：返回字典中元素的个数；
  • keys()：返回字典中所有键名；
  • values()：返回字典中所有键值；
  • entries()：返回字典中所有键值对。

ES6中的Map类

ES6提供了Map类来表示字典。和Set类一样，使用ES6的Map也需要特别注意浏览器兼容性问题。

Map和Object的区别：Map和Object都是存储键值对，但Object的键只能是字符串，而Map的键可以是任意类型的值。

构造函数 new Map([iterable])：传入一个可迭代对象，其中的所有元素（键值对）将被加入到Set中。

和Set类似，Map中的键是唯一的，但判断值是否相等的算法和严格相等不完全一致，主要区别在于NaN !== NaN的结果为true，但在Map中NaN被视为相同的键。

值得注意的是，对于引用类型来说，Map的键实际上是跟内存地址绑定的，只要内存地址不一样，就视为两个键。比如只有对同一个对象的引用，Map才将其视为同一个键。对于简单类型来说，只要两个键严格相等（NaN例外），Map就将其视为同一个键。

所有Map实例都继承自Map.prototype，Map.prototype的属性和方法有：

  • Map.prototype.size：返回Map中元素（键值对）的个数；
  • Map.prototype.get(key)：返回键key对应的值，如果键不存在则返回undefined；
  • Map.prototype.set(key, value)：设置键key对应的值并返回Map对象本身，如果键key已存在则更新值；
  • Map.prototype.has(key)：返回一个布尔值，表示键key是否在Map中；
  • Map.prototype.delete(key)：删除Map中的键key并返回结果；
  • Map.prototype.clear()：移除Map中所有元素；
  • Map.prototype.keys()：返回一个键名的迭代器；
  • Map.prototype.values()：返回一个键值的迭代器；
  • Map.prototype.entries()：返回一个键值对的迭代器；
  • Map.prototype.forEach(callbackFn[, thisArg])：使用回调函数按插入顺序遍历Map中每个元素。
